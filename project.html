Project Showcase


A project that I did with my team at work was configuring a provisioning package for all the windows devices that we install. This allowed us to save time by creating a copy of all the windows settings from a work desktop and apply them to a large amount of laptops and desktops. We also created batch scripts that made it easier to add wifi connections. We saved all provisioning package and the scripts to our USB drives. Once the computer ran the provisioning package, it had downloaded all the applications and settings that the client wanted.
