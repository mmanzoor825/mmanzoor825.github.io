Skill Details


Networking

Computer networking is extremely important in IT. Almost all enterprise environments have computer networks. Some of the topics I first learned included IP addresses, DNS, DHCP, and the differnt types of networking devices like routers and switches.




Command Line

IT professionals often use the command line since it is a powerful environment for troubleshooting. One of the ways that I use the command line is when I have to troubleshoot a computer that is having network issues. I often use the ping command to test reachability to a server or a printer. I am still learning and getting comfortable with the command line. One of the ways I practice getting better at the command line is by running Linux virtual machines on my laptop.

Printers

Printer issues are one of the most common problems people have at work. I have encountered many printer issues on the job. Sometimes Printers have network issues and can't seem to connect to a workstation. Other times there is a hardware issue that interferes with the printing process. Through work experience, I have gotten better at dealing with printer drivers, toner cartidges, and network issues.
